Asteɛṛef-a d win ay yessejhaden ugar tadukli gar uwanak d weɣlan acku aya yekka-d seg lebɣi n tmeddurt ddukkli d lebni n weɣlan ddukkli », ɣef waken ay d-yenna Slimane Hachi deg yiwen n usarag aɣelnaw ɣef Yennayer ay tessuddes tmeghla tadigant n yedles deg ukatar n wesfugel n yiɣef n useggas amaziɣ ay d-yebdan ass n lḥedd deg Tizi Wezzu.
Yenna-d daɣ Mass Hachi dakken « lebɣi-a n tmeddurt d lebni ddukkli d win ay yebnan ɣef llsas aydeg yal yiwen yesteɛṛef s wayeḍ. Mi ara yesteɛṛef uwanak s weɣlan yernu ad yesteɛṛef weɣlan s uwanak, aya ad yeǧǧ talwit tanmettit ad d-tili.
Ɣef leḥsab n Hachi, aɣtas n uselway n tegduda ad yessidef Yennayer deg teswast tunṣibt n tmeɣriwin tiɣelnawin d llsas n ussejhed n wassaɣen inmettiyen. « Aya yessejhad assenfali n tmeddurt ddukkli i lmend n lebni n weɣlan, n uhedden d talwit. Aya d axatar acku yella-d d tiririt i tmetti ay la d-yessuturen acḥal n yiseggasen aya asteɛṛef s twelha-a tamezruyant yellan d llsas n tmagit », ɣef wakken ay d-yerna.
Asfugel n yiɣef n useggas amaziɣ d win ara yesɛun ṣṣifa d tasurfant aseggas-a, imi ay d-yella d taswiɛt n ureṣṣi n llsas, ɣef wakken ay d-yenna unemhal n Wammas-a ay d-yesmektin dakken iɣef n useggas amaziɣ d ass amezwaru n teswast ay sseqdacen deg wakk Tafriqt n Ugafa, yernu iwekked-d dakken « asfugel n Yennayer d tuḍḍfa deg tɣerma, idles d tmagit ».
