Yerna ussan-a kan iɛeddan umḍan n yidlisen n temkarḍit taqbaylit, s tlalit n wungal wis sin n umaru-amedyaz Mhenni Xalifi.
Ungal-a amaynut iwumi yettunefk uzwel "IMEHBAL" d adlis wis ukuẓ n Mhenni Xalifi i d-yessufɣen yakan kraḍ n yidlisen uqbel-is:
"Nnehda n uḥulfu", ammud n yisefra, i d-yeffɣen deg useggas n 2013 s tallelt n Tdukkla Tadelsant Adrar n Fad, "Asdawan", ungal-ines amezwaru i
d-yessaẓreg di 2014, akked "Nna Geltum", ammud n tullisin i d-yeffɣen deg 2016. Yessekles daɣ aḍebsi n ccna "Tifrat", deg 2015.
Dɣa seld asnerni n tekti d kra n wakud i d-yettewten deg leqdic d uraju, yuɣal-d umeskar, mmi-s n temnaḍt n Ayt Smaɛel,
s usufeɣ n wungal-nniḍen ɣur teẓrigin Ccix Muḥend Ulḥusin. Isalli-a, yessekcem tumert deg wulawen n kra n win iḥemmlen tutlayt n Mulud At Mɛemmer,
ama d imura, imedyazen, iselmaden d yakk imeɣnasen d tarwa n lasel, i yeṭṭafaren, ttganin ajdid ara yefken ugar n wazal i wayen yerzan merra taqbaylit,
ama d tasekla neɣ d kra-nniḍen. War ma nettu d akken aṭas seg-sen i yellan d imeɣriyen imekdiyen n tutlayt-nsen tayemmat,
i iḥemmlen ad wansen tirawt talatinit swayes ccɣabayen allaɣen, ssifsusayen ula d iqerra. Ungal "IMEHBAL", deg-s ugar n 285 n yisebtar,
ttwabḍan d iḥricen i d-iḥekkun ɣef kra n temsal d tlufa yemgaraden ara yaf yimeɣri deg teɣzi-s. Deg tenfust, yella wayen i d-yeddan ɣef tmetti,
neɣ ɣef tussna n tmetti, d wamek yezmer ad yennerni yir axemmem armi yuɣal d ugur, d aɣilif ameqqran. Deg-s daɣen, gar cwiṭ d waṭas,
ajbad n ddhen d wallaɣ ɣer kra seg tfelseft n ddunit d tmussniwin i d-yettilin deg tudert n yal ass. Gar sin n yisenfan n tira, yeɛreḍ umeskar,
ad yessemlil aqbur d umaynut, ad yeg assaɣ ijehden gar ṭṭbiɛat yemgaraden n yiwudan d tmeẓla n umalas.
Aya, ad t-yaf yimeɣri seg yisebtar imezwura mi ara yebdu yettifrir-d wungal, yettban-d ugama n wuguren d yiswi-s,
armi d isebtar ineggura ideg ara d-tbin tafat ɣef waṭas n yisental d tɣawsiwin. « Deg wungal-a, i uriɣ iseggasen-a iɛeddan,
ɛerḍeɣ s tirawt ad d-ddmeɣ tisit, ad tt-sekneɣ i tmetti akken ad twali iman-is deg-s. Am wakken rriɣ-t d izen, s wudem n teqsiḍt,
wissen ahat ad yili useggem n waṭas n tɣawsiwin n tudert sɣur wid terza taluft ». I aɣ-d-yenna umeskar n wungal-a, Mass Mhenni Xalifi.
Ad d-nesmekti d akken tzemrem ad tafem ungal-a ussan-a n sdat-neɣ deg tuget n tnedlisin.
